# Visual Explainer

## What It Does
Creates diagrams, visual walkthroughs, and image-backed explanations for code, systems, and workflows.

## When To Use It
Use this when text alone is too dense or when a diagram would clarify structure, flow, or UI behavior.

## How To Invoke It
Reach for the image or diagram generation path already available in the environment, then attach the visual artifact to the explanation.
